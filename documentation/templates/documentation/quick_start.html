3Ô∏è‚É£ Try Your First API Call

REST API Example

List all products using REST:

SOAP API Example

Get product details using SOAP:

curl -X POST http://localhost:8000/inventory/soap/ \
  -H "Content-Type: text/xml" \
  -d '<soapenv:Envelope xmlns:soapenv="http://schemas.xmlsoap.org/soap/envelope/" 
                  xmlns:flip="flipkart.seller.inventory">
   <soapenv:Header/>
   <soapenv:Body>
      <flip:get_product>
         <flip:sku>SKU-1001</flip:sku>
      </flip:get_product>
   </soapenv:Body>
</soapenv:Envelope>'
